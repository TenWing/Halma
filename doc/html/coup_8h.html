Detailed Description

Définit la structure d'un coup, mais aussi d'une liste de coups.

Author
Tendry
Version
1.0
Date
21 novembre 2014

Un coup est considéré comme étant un déplacement de pion, dans le jeu de l'Halma, un pion peut se déplacer plusieurs fois dans un tour en sautant d'autre pions et en enchaînant les sauts, chaque étape intermédiaire est un coup. Une liste de coups est considérée comme une pile de coups, puisque lorsqu'on dépile le dernier l'élément le plus haut de la pile est sensé être l'avant dernier coup.
